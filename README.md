## ` how we work `

Every project moves through 5 stages:

```
  ◯ Ideation  ──▶  ◯ Prototyping  ──▶  ◯ Building  ──▶  ◯ Testing  ──▶  ● Deployed
```

| Stage | Meaning |
|---|---|
| `Ideation` | Concept being defined, research in progress |
| `Prototyping` | First build, proof of concept |
| `Building` | Active development, core features shipping |
| `Testing` | Feature complete, being refined |
| `Deployed` | Live and publicly available |
